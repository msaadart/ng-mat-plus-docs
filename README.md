##  Overview

**ng-mat-plus** is a modular, performance-optimized Angular UI library built on
top of Angular Material. It is designed to accelerate development of large,
real-world applications such as:

- Healthcare Systems
- ERP & Enterprise Platforms
- Admin Panels & Dashboards
- Internal Business Applications

The library follows Angular and Material best practices and is fully compatible
with **Angular 20+ Standalone Components**.

---

##  Key Highlights

- Angular 20+ compatible
- Standalone components & directives
- Modular and tree-shakable architecture
- SCSS-based multi-theme system
- Accessibility-first (a11y compliant)
- Enterprise-ready table solutions
- Performance-focused utilities and services
